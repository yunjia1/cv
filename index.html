LastMile - Mapper Application

Fully functional mapper application designed using the ArcGIS API and C++. Map uses data structures to organize large amounts of data and displays information such as point of interests, features, transit ways, and more. The map works for different cities and is compatible with different languages. Navigation is also possible with the map using Dijkstra and A* algorithms. The path can be visualized on the map and guided through detailed directions display.

Wrapped Anytime

Project which displays a certain Spotify users “wrapped” including their top artists and songs over time (recent, over last 6 months, and all time). Application also provides suggested playlists based on the user’s most listened to genres. Project uses Python and the Spotify API.
